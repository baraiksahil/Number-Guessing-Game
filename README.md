# 🎮 Number Guessing Game

A classic command-line based Number Guessing Game built in C++ to practice basic control flow, random number generation, input/output streams, and game loops.

## ✨ Features

🎲 **Random number generation** (1 to 100) using `rand()` and time-based seeding

💡 **Interactive feedback** ("Too high!" or "Too low!") to guide the player

📊 **Attempt tracking** to show how many guesses it took to find the correct number

🔄 **Replay functionality** allowing players to start a new round immediately

## 🛠️ Tech Stack

- C++
- Standard Library (iostream, cstdlib, ctime)

## 📚 What I Learned

- Standard input/output streams (`std::cin`, `std::cout`)
- Generating pseudo-random numbers using `srand(time(0))` and `rand()`
- Game loops and conditional logic (`do-while` and `while` loops)
- State management for tracking game statistics (number of attempts)
- Input flow control and conditional branching
